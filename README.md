# proyecto_c_2026
Conversor de Monedas

## Descripción
La tarea realizada es un programa desarrollado en el Lenguaje C el cual realiza conversión de monedas de Quetzales a Dolares y Euros, de Dolares a Quetzales, y de Euros a Quetzales

## Prerrequisitos 
- Instalación de sistema operativo Windows o Linux
- Compilador GCC

## Pasos para la Ejecución
- Abrir la terminal
- Ubicarse en la carpeta del proyecto
- Compilar el programa usando: gcc conversor.c -o conversor
- Ejecutar el programa: ./conversor
